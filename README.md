# CODSOFT-task1-# Rule-Based Chatbot using Python 🤖

## Overview

This project implements a simple **Rule-Based Chatbot** using Python.
The chatbot responds to user inputs using predefined rules and pattern matching.
It demonstrates basic concepts of **Natural Language Processing (NLP)** and **conversation flow**.

The chatbot can interact with users by recognizing keywords and returning appropriate responses.

## Features

* Responds to basic greetings like *Hello, Hi, Hey*
* Answers simple questions such as *How are you?* and *What is your name?*
* Provides **current time and date**
* Gives help information to users
* Ends conversation when the user types **bye**, **exit**, or **quit**
* Uses **pattern matching and rule-based logic**

## Technologies Used

* Python
* Regular Expressions (re)
* Random module
* Datetime module

## How It Works

1. The user enters a message.
2. The chatbot checks the message against predefined patterns.
3. If a pattern matches, the chatbot returns a corresponding response.
4. If no pattern matches, the chatbot provides a default response.

## Example Conversation

User: Hello
Bot: Hi there! How can I assist you?

User: What is your name
Bot: I am a rule-based chatbot built using Python.

User: What time is it
Bot: Current time is 14:30:21

User: Bye
Bot: Goodbye! Have a great day.

## How to Run the Project

1. Install Python on your system.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run the following command:

python chatbot.py
